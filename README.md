# CS2 Case Opening EV Analysis

Expected value analysis of CS2 case openings using real-time price data.

## What's inside
- Full EV calculation for 5 cases (Kilowatt, Revolution, Fracture, Glove, Chroma 2)
- Trade-Up contract analysis (5x Covert → Knife)
- Price data sourced from Skinport API (April 2026)
- Drop rates from Valve's official 2017 disclosure

## Key findings
- Kilowatt Case (newest): 15.8% return rate
- Fracture Case: 288.5% return rate  
- Trade-Up: 61.9% return — worse than buying a knife directly
